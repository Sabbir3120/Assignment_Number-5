What is the Document Object Model?

The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

  • Using querySelector
  • Using querySelectorAll
What is event delegation in the context of the DOM and why is it usefull?

Event delegation is a technique that involves leveraging event bubbling to manage events at a higher level in the DOM rather than attaching event listeners to each element.
